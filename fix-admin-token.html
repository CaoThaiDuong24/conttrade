🚨 Admin Token Hết Hạn

❌ Vấn Đề

Bạn đang dùng admin account nhưng bị yêu cầu đăng nhập lại khi truy cập trang cập nhật quyền.

Nguyên nhân:

  • Token cũ đã hết hạn (JWT expires sau 7 ngày)
  • Hoặc token được tạo trước khi chúng ta fix permissions_updated_at logic

📋 Hoặc Làm Thủ Công:

Cách 1: Sử Dụng Console

Mở DevTools (F12) và chạy:

document.cookie = "accessToken=; path=/; max-age=0";
location.href = "http://localhost:3000/vi/login";

Cách 2: Xóa Cookie Thủ Công

  1. Mở DevTools (F12)
  2. Vào tab Application → Cookies → localhost:3000
  3. Click phải vào cookie accessToken → Delete
  4. Refresh trang (F5)
  5. Đăng nhập lại

Cách 3: Hard Refresh

Nhấn Ctrl + Shift + R để xóa cache và reload

🔍 Debug: Kiểm Tra Token Hiện Tại

Để xem token hiện tại có gì:

const token = document.cookie.split('; ').find(row => row.startsWith('accessToken='))?.split('=')[1];
if (token) {
  const payload = JSON.parse(atob(token.split('.')[1]));
  console.log('Token expires:', new Date(payload.exp * 1000));
  console.log('Permissions:', payload.permissions?.length);
} else {
  console.log('No token found');
}
